# Introduction

Welcome to the **ProxyPay** developer documentation portal.

This site is built with [Docusaurus](https://docusaurus.io/). The home page
(["Scroll-Spy Demo"](pathname:///)) showcases a sticky navigation bar with
active-section highlighting powered by the `IntersectionObserver` API.

## What's here

- A sticky in-page section nav that highlights the active section as you scroll.
- Smooth-scroll navigation when a section link is clicked.
- WCAG 2.1 AA compliant active / inactive link styling.

See [`src/components/ScrollSpyNav`](https://github.com/Saharadigitals256/proxypay)
for the implementation.
